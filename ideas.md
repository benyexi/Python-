# Python 代码执行器 - 设计方案

## 三个设计方向

### 方向一：Terminal Noir
- **简介**：深色终端美学，致敬经典黑客/开发者终端体验，用荧光绿和深邃黑色营造专业编程氛围。
- **概率**：0.04

### 方向二：Brutalist Code Lab
- **简介**：粗犷主义实验室风格，大胆的几何分割、单色调配合高对比度强调色，突出功能性和工具感。
- **概率**：0.06

### 方向三：Soft IDE
- **简介**：柔和现代的 IDE 风格，借鉴 VS Code 的布局逻辑但用更温暖的配色和圆润的设计语言，让代码编辑变得舒适。
- **概率**：0.03

## 选定方向：Terminal Noir

### Design Movement
Cyberpunk Terminal Aesthetic — 融合复古终端界面与现代极简主义，创造沉浸式编程体验。

### Core Principles
1. **深色沉浸**：以近乎纯黑的背景为基底，减少视觉干扰，让代码成为焦点
2. **荧光点缀**：用电子绿/青色作为高亮色，模拟 CRT 显示器的发光效果
3. **等宽至上**：所有代码区域使用精心选择的等宽字体，非代码区域用无衬线体形成层次
4. **即时反馈**：每个操作都有清晰的视觉反馈，模拟终端的即时响应感

### Color Philosophy
- **主背景**：`#0a0e14` — 深空黑，比纯黑更有层次
- **次背景**：`#1a1f2e` — 用于卡片和面板区分
- **主强调色**：`#00ff9f` — 电子绿，代表执行/成功/活跃状态
- **次强调色**：`#00b8ff` — 冷蓝色，用于交互元素和链接
- **警告色**：`#ff6b6b` — 柔和红，用于错误输出
- **文本色**：`#e0e0e0` — 高对比度但不刺眼的浅灰

### Layout Paradigm
垂直分割布局 — 上方为代码输入区（占据主要空间），下方为输出/控制台区域，模拟真实终端的上下分屏体验。侧边有一个窄工具栏提供快捷操作。

### Signature Elements
1. **发光边框**：活跃区域带有微弱的荧光绿 glow 效果
2. **扫描线纹理**：极淡的水平线纹理覆盖在背景上，致敬 CRT 显示器
3. **打字机光标**：自定义的闪烁光标动画

### Interaction Philosophy
所有交互模拟终端操作的即时性 — 按钮点击有"按下"反馈，代码执行有加载脉冲动画，输出以打字机效果逐行显示。

### Animation
- 代码执行时：运行按钮发出脉冲光效（pulse glow）
- 输出显示：轻微的逐行淡入效果（50ms stagger）
- 面板切换：快速滑动过渡（180ms ease-out）
- 光标闪烁：经典的 step 动画，非平滑过渡

### Typography System
- **代码字体**：JetBrains Mono — 专为编程设计的等宽字体，连字符优美
- **UI 字体**：Space Grotesk — 几何无衬线体，带有科技感
- **层次**：标题用 Space Grotesk Bold，正文用 Regular，代码区全部 JetBrains Mono

### Brand Essence
一句话定位：为开发者打造的即时 Python 执行环境，快速、纯粹、无干扰。
三个性格形容词：**极客的**、**高效的**、**沉浸的**

### Brand Voice
标题和文案风格：简洁、直接、带有技术幽默感。
- 示例 1："Write. Run. Done." （主标语）
- 示例 2："Your code, executed in milliseconds." （副标语）

### Wordmark & Logo
一个抽象的 Python 蛇形符号与终端光标 `>_` 的融合图形，用电子绿色呈现。

### Signature Brand Color
**Electric Green `#00ff9f`** — 这个颜色是品牌的核心标识，代表"执行"和"活跃"。
